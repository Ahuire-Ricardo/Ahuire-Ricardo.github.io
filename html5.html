Every HTML element that you access via JavaScript has a style object. This object allows you to specify a CSS property and set its value.

Also, when you set this property on an element, you override and erase any styles that have been set elsewhere for that element's particular property you are setting. Setting the border property, for example, will override settings made elsewhere for that element's border property in the head section, or external style sheets.
