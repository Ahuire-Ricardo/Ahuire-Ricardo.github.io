Conditional and Parameters

This program compares two numbers and determines if it is greater, less or equal by means of a conditional sentence.

	
function obtainNumber(favoriteNumber, guess) {
  var number = ""; 

  if (guess > favoriteNumber) {
    number = "Too high";
  }
  else if (guess == favoriteNumber) {
     number = "You got it!";
  }
  else {
     number = "Too low";
  }

  document.getElementById("result2").innerHTML = number;
}
