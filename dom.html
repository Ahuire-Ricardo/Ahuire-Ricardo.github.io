Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects . That way, programming languages can connect to the page.

The DOM is not a programming language, but without it, the JavaScript language wouldn't have any model or notion of web pages, HTML documents, XML documents, and their component parts (e.g. elements).

Storing and Retrieving Arrays

	
// our array
var characters = ["Luke", "Leia", "Yoda","Rey","Han Solo","Chewbacca","R2-D2", "Darth Vader"];

function getArray(){

	// storing our array as a string
	localStorage.setItem("Starwars", JSON.stringify(characters));

	// show json object
	document.getElementById("results1").innerHTML = 'Json File: ' + localStorage.getItem("Starwars");

	// retrieving our data and converting it back into an array
	var retrievedData = localStorage.getItem("Starwars");
	var characters2 = JSON.parse(retrievedData);

	//making sure it still is an array
	document.getElementById("results2").innerHTML = 'Length Array: ' + characters2.length;
	document.getElementById("results3").innerHTML = 'Array: ' + characters2;


}
