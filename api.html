Local Storage API

With web storage, web applications can store data locally within the user's browser. Before HTML5, application data had to be stored in cookies(text file), included in every server request. Web storage is more secure and provides two objects for storing data on the client:

  • window.localStorage - stores data with no expiration date.
  • window.sessionStorage - stores data for one session (data is lost when the browser tab is closed).

Storing and Retrieving Simple Data



  function getName(){
	// Check browser support
	if (typeof(Storage) !== "undefined") {
	    // Store
	    localStorage.setItem("lastname", "Ahuire");
	    // Retrieve
	    document.getElementById("result").innerHTML = localStorage.getItem("lastname");
	} else {
	    document.getElementById("result").innerHTML = "Sorry, your browser does not support Web Storage...";
	}
   }
